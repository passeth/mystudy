Reference · Glossary

Loop / Graph Engineering 용어

이 코스에서 우선하는 정의. 에이전트 실행 그래프와 혼동되는 항목은 명시한다.

Loop engineering
측정→목표 대비→조치→반복의 자기개선 사이클을 설계·운영하는 기술.
Four-stroke loop
Control, Reference, Gap, Act.
Goodhart’s law
지표가 목표가 되면 좋은 지표이기를 멈춘다.
Blindness upward
루프가 레퍼런스(목표)의 타당성을 물을 수 없는 구조.
Loop conflict
독립적으로 잘 도는 루프들이 시스템 수준에서 상충.
Measurement decay
측정이 현실과 끊기거나 서류 대조로 전락.
Graph engineering (Perez)
루프들을 감시·공급·거부권 엣지로 연결하는 루프 아키텍처.
Pairing
최적화 지표 + 반지표 감시 루프.
Hierarchy of loops
느린 루프가 빠른 루프의 목표를 소유·개정.
Arbitration
충돌 루프 위 트레이드오프 소유 루프.
Audit loop
다른 루프의 현실 접촉을 검사하는 루프.
Anchor
논쟁 여지 적은 현실 접촉 측정.
Frozen rule
최적화 루프가 변경 불가인 규칙(예: held-out 분리).
Grounded vs ungrounded
개선 기계가 현실에 닿는지 여부. Perez가 제시하는 더 깊은 축.
Agent execution graph
(구분) LangGraph 등 실행 노드·엣지·상태 공유 토폴로지. 개선 루프 그래프와 다른 층.
Champion–challenger
후보가 현역을 라이브에서 이겨야 교체되는 배치 루프 패턴.
